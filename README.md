# invespro

Open source investment risk profiling engine for Node.js and React.
Part of the [vibedcoder](https://github.com/vibedcoder) fintech toolkit.

## Packages

| Package                      | Description                  |
| ---------------------------- | ---------------------------- |
| `@vibedcoder/invespro-types` | Domain types and Zod schemas |
| `@vibedcoder/invespro-core`  | Risk profiling engine        |
| `@vibedcoder/invespro-hono`  | Hono REST adapter            |
| `@vibedcoder/invespro-cli`   | CLI tool                     |

## Development

```bash
pnpm install
pnpm -r build
pnpm -r test
```
