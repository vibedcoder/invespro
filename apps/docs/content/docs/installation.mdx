---
title: Installation
description: Install the packages you need for embedded, REST, or CLI usage.
---

Install only the package required by your integration surface.

## Core engine

```bash
pnpm add @vibedcoder/invespro-core
```

Use this when you want to evaluate applicants inside an existing Node.js service.

## REST adapter

```bash
pnpm add @vibedcoder/invespro-hono hono
```

Use this when another system should call Invespro over HTTP.

## CLI

```bash
pnpm add -g @vibedcoder/invespro-cli
```

Use this for local files, operations workflows, or quick checks from JSON and CSV.

## Types

```bash
pnpm add @vibedcoder/invespro-types
```

You usually do not need to install this directly unless you want the schemas
without the engine.

## Runtime

The packages currently target modern Node.js. Use Node.js 24 or newer for the
same runtime assumptions used by the published packages and demo.
