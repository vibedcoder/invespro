Install only the package required by your integration surface.

## Core engine

<CodeBlockTabs defaultValue="pnpm">
  <CodeBlockTabsList>
    <CodeBlockTabsTrigger value="npm">npm</CodeBlockTabsTrigger>
    <CodeBlockTabsTrigger value="pnpm">pnpm</CodeBlockTabsTrigger>
    <CodeBlockTabsTrigger value="yarn">yarn</CodeBlockTabsTrigger>
    <CodeBlockTabsTrigger value="bun">bun</CodeBlockTabsTrigger>
  </CodeBlockTabsList>

  <CodeBlockTab value="npm">

```bash
npm install @vibedcoder/invespro-core
```

  </CodeBlockTab>

  <CodeBlockTab value="pnpm">

```bash
pnpm add @vibedcoder/invespro-core
```

  </CodeBlockTab>

  <CodeBlockTab value="yarn">

```bash
yarn add @vibedcoder/invespro-core
```

  </CodeBlockTab>

  <CodeBlockTab value="bun">

```bash
bun add @vibedcoder/invespro-core
```

  </CodeBlockTab>
</CodeBlockTabs>

Use this when you want to evaluate applicants inside an existing Node.js service.

## Runtime

The packages currently target modern Node.js. Use Node.js 24 or newer for the
same runtime assumptions used by the published packages and demo.
